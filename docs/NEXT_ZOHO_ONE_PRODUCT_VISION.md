# Next Zoho One Product Vision

Date: 2026-06-04

## 1. CEO Product Vision

Build the operating system for Indian and emerging-market businesses: one platform where customers, employees, projects, money, approvals, documents, and AI decisions live in the same record graph. The product should begin with HRMS, CRM, and PMS, then expand into vertical editions and finance/inventory/accounting integrations.

## 2. CTO Architecture Vision

The architecture should be event-driven, workflow-native, AI-governed, mobile-aware, and integration-ready. Core primitives: canonical entities, event bus, workflow engine, audit ledger, connector framework, AI evidence layer, role/security policy engine, reporting semantic layer, offline sync service, and tenant configuration studio.

## 3. Product Roadmap

| Horizon | Theme |
|---|---|
| 0-6 months | Commercial suite hardening: workflows, approvals, mobile, CRM/PMS integration |
| 6-18 months | Vertical editions, embedded AI, billing/invoice handoff, BI dashboards |
| 18-36 months | ERP expansion, accounting/inventory/procurement integrations, partner console |
| 36-60 months | Low-code platform, marketplace, autonomous AI operations, industry intelligence |
