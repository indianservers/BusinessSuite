# Implementation Roadmap

Date: 2026-06-04

## Roadmap Strategy

Build the next roadmap around marketable workflows, not isolated pages. The fastest path to competitiveness is:

1. Make CRM and PMS feel production-integrated.
2. Add automation and AI where users make daily decisions.
3. Harden scale, audit, integrations, and mobile.
4. Add suite-level lead-to-cash and project-to-invoice workflows.

## 0-30 Days: Commercial MVP Hardening

| Priority | Item | Outcome |
|---|---|---|
| P0 | Module startup validator and smoke tests | CRM/PMS/HRMS cannot silently disappear |
| P0 | CRM email/calendar setup status | Users know what is configured vs mock/logging |
| P0 | PMS project templates/cloning | Faster customer onboarding |
| P0 | Suite approval inbox | Immediate manager productivity |
| P1 | CRM web-to-lead forms | Inbound capture |
| P1 | PMS recurring tasks/reminders | Operational reliability |

| P1 | Mobile PWA smoke coverage for approvals | Manager adoption |
